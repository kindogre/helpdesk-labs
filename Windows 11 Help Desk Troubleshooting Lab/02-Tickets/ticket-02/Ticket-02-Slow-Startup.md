# Ticket 02 — Slow System Startup

## Ticket ID
HD-02

## Scenario
User reports that the computer takes a long time to start and becomes slow after login.

## User Impact
Delayed access to applications and reduced productivity due to slow system performance.

## Symptoms Observed
- Slow login time
- Applications take time to open after startup
- System feels sluggish immediately after boot

## Initial Checks
- Verified system resources
- Checked startup applications
- Observed system performance after login

## Troubleshooting Steps Performed
1. Logged into system and observed slow startup behavior
2. Opened Task Manager and navigated to Startup Apps
3. Identified applications with high startup impact
4. Disabled non-essential startup programs
5. Restarted system to test performance

## Root Cause
Excessive startup applications consuming system resources during boot.

## Resolution
Disabled unnecessary startup applications to improve system performance.

## Verification
- System startup time improved
- Applications opened faster after login
- Overall responsiveness increased

## Screenshots
- Before: slower startup
- During: disabled some startup apps
- After: normal startup
