# Windows 11 Help Desk Lab Setup

## Lab Objective
Build a Windows 11 virtual machine to simulate common help desk issues and practice troubleshooting in a structured, documented way.

## Host System
- Device: MacBook with M4 chip
- Hypervisor: VMware Fusion Pro

## Virtual Machine Details
- Guest OS: Windows 11
- VM Name: HELPDESK-PC01
- Computer Name inside Windows: HELPDESK-PC01
- vCPU: 2
- Memory: 4 GB
- Disk Size: 64 GB
- Network Mode: NAT

## User Accounts Created
- LocalAdmin — Administrator
- TestUser1 — Standard User

## Installed Software
- Google Chrome
- 7-Zip
- CCleaner
- PowerToys

## Windows Configuration Completed
- Renamed PC
- Created local accounts
- Installed common applications
- Fully updated Windows
- Verified internet access

## Snapshot Information
- Snapshot Name: Clean Windows 11 Base
- Purpose: Baseline restore point before starting each troubleshooting scenario

## Lab Notes
This setup creates a repeatable Windows 11 help desk lab environment for testing common desktop support issues.
